When an image is removed from the Froala Rich Text Editor, the image is not from the server. In order to delete an image from the server, you have to use the editor events to and make a delete request to the server.

Image Delete Flow

  1. Include the image plugin file.

    Image is a plugin for the editor and the image.min.js file must be included before being able to use it.

  2. Set the editor's image events.

    There are 2 events related to image remove: image.beforeRemove and image.removed. The one that is of interest for us is image.removed, because it is triggered right after the image is removed from the WYSIWYG HTML editor.
